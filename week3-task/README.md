# # week three project :serverless
build a serverless function triggered by HTTP API Gateway

# # Project Overview
The rationale behind this project is to build a modern,scalable application architecture without the overhead of managing physical or virtual servers.

# # Tech stacks
AWS Lambda - it forces you to write stateless code which is a fundamental principle of modern cloud engineering.
API Gateway - Handles authentication, rate limiting, and routing before the request touches the code.
